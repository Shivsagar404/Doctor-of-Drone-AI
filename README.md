# 🚁 Doctor of Drone AI

### Your Intelligent Drone Health & Troubleshooting Assistant

Doctor of Drone AI is an AI-powered drone diagnostic assistant designed to identify, analyze, and troubleshoot common drone faults using Machine Learning and Natural Language Processing (NLP).

Just like a doctor diagnoses a patient, Doctor of Drone AI analyzes drone symptoms, predicts possible faults, and recommends corrective actions to help pilots maintain safe and reliable flight operations.

---

## 🩺 Key Features

### 🤖 AI-Based Fault Diagnosis

Predicts drone faults using a trained Machine Learning model.

### 🧠 Natural Language Understanding

Users can describe drone issues in plain English.

Example:

"My drone rotates right after takeoff and GPS is not locking."

### 🔍 Smart Fault Detection

Identifies common drone issues:

* Compass Calibration Error
* GPS Failure
* ESC Fault
* Motor Direction Error
* Propeller Damage

### 🛠 Repair & Troubleshooting Suggestions

Provides maintenance guidance and recommended solutions.

### 🖥 Interactive Desktop Application

User-friendly GUI developed using Python Tkinter.
